This is a example of the scoped-css.js

How to scope your styles?

The scoped attribute will automatically scope your css to the appropriate parent.

<style scoped>
h1 {
color: white;
}
</style>

Alternatively

You can set the attribute scoped to whatever you like. For example:

<style scope="#test">
p {
font-weight: bold;
}
</style>

Scope parent

You can also use this to scope the parent element it self using #this

<style scoped>
#this {
background: black;
}
</style>
